# Architecture Decision Records

Architecture decisions are recorded using numbered Markdown documents.

Initial decisions will cover:

1. Modular monolith
2. Django and Django REST Framework
3. PostgreSQL as system of record
4. Projects and deployment environments
5. Celery and Redis
6. Transactional outbox
7. OpenTelemetry-compatible ingestion
8. Versioned datasets and evaluators
